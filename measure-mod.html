Measurement is more than just the assignment of numbers according to rules (i.e., labeling); it is a process by which an attempt is made to understand the nature of a variable (Bridgman, 1928)

Thus, a latent variable model is simply a model that relates the expectation of observables to a latent structure through some regression function. (Borsboom, 2008, p. 27)

Measurement invariance / Differential item functioning

Types of measurement invariance

Type/Levels Description
Configuralinvariance (baseline) the same variables define each latent variable across groups/time points
Weak factorial invariance restricted factor loadings across groups/timepoints
Strong factorial invariance restricted factor loadings and intercepts across groups/time points
Strict factorial invariance restricted factor loadings, intercepts and error variances across groups/time points
